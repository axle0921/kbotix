로그인

KBOTix 회원은 로그인을 통해 구매 또는 판매를 할 수 있으며 다음의 경우 로그인이 제한된다.

  • 로그인 5회 연속 실패하여 일시 차단된 경우
  • 벌점 기준에 따라 현재 '불량회원'인 경우
  • 신고 기준에 따라 현재 '불량회원'인 경우

Manual

  • 'ID 저장' 선택 후 정상적으로 로그인되면 30일간 해당 PC에 ID값이 저장되어 ID 입력란에 자동으로 입력된다
  • 로그인 필수 URL 접근 시 인증 후 입력한 URL로 이동한다
  • 로그인 5회 연속 실패 시 인증 일시 차단된다
  • 벌점 기준에 따라 현재 '불량회원'인 경우 로그인이 제한된다
  • 신고 기준에 따라 현재 '불량회원'인 경우 로그인이 제한된다

Out of Scope

  • 로그인 인증 보안
